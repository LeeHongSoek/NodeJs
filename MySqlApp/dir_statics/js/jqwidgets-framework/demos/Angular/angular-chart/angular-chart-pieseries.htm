This is an example of Angular 8 Pie chart. Pie charts are often used to show the relative size of the measured data. The data is prepared as an array from a CSV file. The type of the seriesGroup is pie. You can see how to set the position of the legend with the legendPosition setting. You can also see how to set the radius, the label of the radius and the initial angle of the pie chart. The center offset is set to 0 in this case.
