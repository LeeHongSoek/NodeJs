This is an example of Angular 8 DataTable. The Angular Data Table was built to allow you to easily replace your existing HTML Tables. It can read and display the data from your HTML Tables, but it can also display data from different data sources like XML, JSON, Array, CSV or TSV. The DataTable enables editing of cells and entire rows including options to edit with a dialog or to lock rows. It also supports sorting, filtering and paging. One of the advanced features is that you can use the data table to enable nested tables. Aggregates and grouping are supported as well. You can export your data to Excel, XML, CSV, TSV, JSON, PDF and HTML. Conditional formatting will enable you to add custom CSS styles. Localization will give you the option to show information in other languages than English.
